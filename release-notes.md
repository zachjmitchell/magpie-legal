# Magpie — What's New

## Version 1.0
The first release of Magpie 🐦

- One shared calendar for family, friends, and housemates
- Colour-coded calendars and labels
- Month, week, and day views
- Event comments, countdowns, and reminders
- Repeating events
- Import from Apple Calendar
- Home screen widget
- Magpie Plus: unlimited calendars, members, countdowns, colours, and a
  personalised background

_This page is updated with each release._
